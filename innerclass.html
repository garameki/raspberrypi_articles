innerクラスからouterクラスメソッドをcall

名前空間を汚したくないばっかりに、innerクラスを用いた結果、innerクラスからは、outerクラスのインスタンスメソッドが呼べない件。を解決した。

コード

サンプルソース 
#!/user/bin/env python3

class A:
	def __init__(self):
		self.lcd = 6666
		self.b = self.B()
		self.b.parent = self

	class B:
		def __init__(self):
			pass
		def send(self):
			self.parent.send()

	def send(self):
		print(self.lcd)

a = A()
a.send()
a.b.send()
実行結果 
$ python3 sample.py 
6666
6666
